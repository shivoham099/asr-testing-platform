How to Create Your CSV File:
Step 1: Open Excel/Google Sheets
  • Open Microsoft Excel or Google Sheets
  • Create a new document
Step 2: Add Crop Names
  • Type one crop name in each row
  • Use Column A only
  • Don't add headers or titles
Step 3: Save as CSV
  • Go to File → Save As
  • Choose "CSV" format
  • Make sure encoding is UTF-8
Step 4: Upload Here
  • Click "Choose file" below
  • Select your CSV file
  • Click "Upload & Start Testing"

Important Notes for QA Team:
  • One crop name per row - no commas or separators
  • No header row - start directly with crop names
  • UTF-8 encoding - save your file as UTF-8 to support all languages
  • File extension - must be .csv
  • Language match - make sure your CSV contains crop names in the selected language
Quick Start Guide for QA:
For Hindi Testing:
