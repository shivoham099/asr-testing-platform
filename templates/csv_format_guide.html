CSV Format Guide for QA Team

Quick Instructions:

Choose any of the 4 formats below. All formats work perfectly with the platform!

Format 1: Simple List (Recommended)

Best for: Quick testing, beginners

How to create:

  1. Open Excel/Google Sheets
  2. Type one crop name in each row
  3. Use Column A only
  4. Save as CSV
Example:
गेहूं
चावल
मक्का
बाजरा
ज्वार
रागी
अरहर
चना
मूंग
उड़द
Format 2: Numbered List

Best for: Organized testing, tracking progress

How to create:

  1. Open Excel/Google Sheets
  2. Type numbers in Column A (1, 2, 3...)
  3. Type crop names in Column B
  4. Save as CSV
Example:
1,गेहूं
2,चावल
3,मक्का
4,बाजरा
5,ज्वार
6,रागी
7,अरहर
8,चना
9,मूंग
10,उड़द
Format 3: With Categories

Best for: Detailed testing, crop categorization

How to create:

  1. Open Excel/Google Sheets
  2. Type crop names in Column A
  3. Type categories in Column B (optional)
  4. Save as CSV
Example:
गेहूं,अनाज
चावल,अनाज
मक्का,अनाज
बाजरा,अनाज
टमाटर,सब्जी
आलू,सब्जी
प्याज,सब्जी
गाजर,सब्जी
केला,फल
आम,फल
Format 4: With Descriptions

Best for: Complex testing, detailed analysis

How to create:

  1. Open Excel/Google Sheets
  2. Type crop names in Column A
  3. Type descriptions in Column B (optional)
  4. Save as CSV
Example:
गेहूं,सर्दियों की फसल
चावल,मुख्य अनाज
मक्का,पीला अनाज
बाजरा,छोटे दाने
ज्वार,लंबे पौधे
रागी,पोषक अनाज
अरहर,दाल
चना,प्रोटीन युक्त
मूंग,हरी दाल
उड़द,काली दाल
Important Notes for All Formats:
  • Column A is always used - crop names must be in first column
  • UTF-8 encoding - save your file as UTF-8
  • No headers needed - start directly with data
  • One crop per row - don't put multiple crops in one row
  • File extension - must be .csv
  • Language match - use crop names in selected language
  • No special characters - avoid commas in crop names
  • Test with sample - try with 5-10 crops first

Pro Tips
  • Start small - test with 10-20 crops first
  • Use Format 1 - simplest and most reliable
  • Check encoding - make sure UTF-8 is selected
  • Backup your files - keep copies of your CSV files
  • Test upload - try uploading before starting session
