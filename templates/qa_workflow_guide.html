QA Workflow Guide - How to Use the Platform

Important for QA Team:

Follow this step-by-step guide to ensure proper testing and data collection. Keep your own records as backup!

Complete QA Workflow:
1
Login with Sarvam ID
  • Click "Sign in with Google"
  • Use your Sarvam email (e.g., yourname@sarvam.ai)
  • You'll be redirected to language selection
2
Select Language
  • Choose your testing language: Hindi, Malayalam, Gujarati, Odia, or English
  • Click on the language button
  • You'll be taken to the CSV upload page
3
Upload CSV File
  • Create your CSV file with crop names (one per row)
  • Click "Choose file" and select your CSV
  • Click "Upload & Start Testing"
  • You'll see the first crop name to test
4
Use Words in Sentences
  • Read the crop name displayed on screen
  • Use it in a sentence (e.g., "I am saying टमाटर" for tomato)
  • Don't just say the word alone - use it in context
  • Speak clearly and at normal pace
5
Record Audio
  • Click "Start Recording" (green button with microphone)
  • Say your sentence with the crop name
  • Click "Stop Recording" (red button with square)
  • Wait for transcription to appear
  • Check if the crop name was detected correctly
6
Submit and Record Again
  • Click "Submit" to save the recording
  • Record the same crop name 4 more times (5 total attempts)
  • Each attempt should be a different sentence
  • Example sentences: "I am saying टमाटर", "The crop is टमाटर", "टमाटर is red", etc.
7
End Session and Create Reports
  • You can click "End Session" anytime to finish testing
  • This will take you to the results page
  • You'll see summary statistics and detailed results
  • Click "Download CSV" to get the complete report
8
Download Reports and Compile Analysis
  • Download the CSV report for each session
  • Compile all reports for final analysis
  • Note which crop names had poor recognition
  • Identify patterns in mispronunciations
Important Backup Instructions
Keep Your Own Records!

In case the platform stops recording data, maintain your own backup:

  • Record mispronounced words in a separate document
  • Note which attempts failed for each crop
  • Keep track of session dates and languages tested
  • Save screenshots of important results
  • Download reports immediately after each session
Pro Tips
  • Test in quiet environment - no background noise
  • Use consistent microphone - same device for all tests
  • Speak naturally - don't over-enunciate
  • Take breaks - don't rush through all crops
  • Test different times - morning vs evening
Quality Checklist:
  • ✅ Used Sarvam email for login
  • ✅ Selected correct language
  • ✅ Uploaded properly formatted CSV
  • ✅ Used crop names in sentences
  • ✅ Recorded 5 attempts per crop
  • ✅ Downloaded CSV reports
  • ✅ Kept backup records
  • ✅ Compiled final analysis
